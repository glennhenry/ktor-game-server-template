About

This command tool enable interaction with the server during runtime.

By implementing the Command interface, you can perform various server-side operations

such as scanning local files, modifying data, triggering tasks, and more.

All of these actions can be executed directly with commands,

without changing the serverâ€™s code or restarting the server, only implementing the command themselves.

Usage

You have to enter command ID followed by its arguments separated by space.

Failing to enter sufficient argument will fail.

More arguments than expected will be ignored.

Leading and trailing spaces are ignored.

Ensure type matches expectation.

Some types are converted automatically (e.g., number and boolean may be captured as string).

A correct command syntax: 'give playerABC water 100'

Use Ctrl+F to search command.
